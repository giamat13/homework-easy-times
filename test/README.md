# 📚 מערכת ניהול שיעורי בית (Homework Management System)

מערכת אינטרנטית נקייה, מודרנית וידידותית למשתמש לניהול משימות לימודיות, מעקב אחר דד-ליינים וארגון חומרי לימוד לפי מקצועות.



## ✨ תכונות עיקריות

* **ניהול מקצועות מותאם אישית:** הוספת מקצועות עם בחירת צבעים ייחודית לזיהוי ויזואלי מהיר.
* **מעקב משימות חכם:** הוספת שיעורי בית הכוללים כותרת, תיאור, תאריך הגשה ורמת עדיפות.
* **ניהול קבצים:** אפשרות לצרף קבצים למשימות ושמירתם בזיכרון המקומי.
* **סטטיסטיקה בזמן אמת:** לוח בקרה המציג את סך המשימות, משימות שהושלמו, משימות ממתינות ומשימות דחופות.
* **מערכת התראות ויזואלית:** סימון אוטומטי של משימות דחופות (פחות מ-48 שעות) או משימות באיחור.
* **ארכיון חכם:** העברת משימות שהושלמו לארכיון לאחר תאריך ההגשה לשמירה על סדר בעיניים.
* **שמירה מקומית:** הנתונים נשמרים בדפדפן (LocalStorage), כך שאין צורך בשרת או בהתחברות.

---

## 🛠 טכנולוגיות

הפרויקט נבנה כ-Single Page Application (SPA) תוך שימוש בטכנולוגיות הבאות:

* **HTML5 & CSS3:** עיצוב רספונסיבי ושימוש ב-Flexbox/Grid.
* **Vanilla JavaScript:** ניהול לוגיקה, מניפולציית DOM וטיפול בקבצים (Base64).
* **Lucide Icons:** עבור האייקונים במערכת.
* **LocalStorage API:** לשמירת נתונים בצד הלקוח.

* **מדווחים (Issue Reporters):** תודה לכל המשתמשים שפתחו **Issues**, דיווחו על באגים והציעו שיפורים שהפכו את המערכת ליציבה יותר.

## 🚀 איך להשתמש?

1.  **הוספת מקצועות:** התחילו בהוספת המקצועות שאתם לומדים ובחרו להם צבע.
2.  **יצירת משימה:** מלאו את פרטי שיעורי הבית, בחרו את המקצוע והגדירו תאריך יעד.
3.  **צירוף קבצים:** ניתן להעלות מסמכים או תמונות הרלוונטיים למשימה.
4.  **מעקב:** סמנו "V" על משימות שבוצעו. המערכת תעדכן את הסטטיסטיקה באופן אוטומטי.

# 📚 מערכת ניהול משימות (Homework Management System)

מערכת אינטרנטית נקייה, מודרנית וידידותית למשתמש לניהול משימות לימודיות, מעקב אחר דד-ליינים, ניהול מבחנים וארגון חומרי לימוד לפי מקצועות. כל הנתונים מסונכרנים בזמן אמת ל-Firebase עם גיבוי מקומי.

---

## ✨ תכונות עיקריות

* **🎓 מצבי שימוש:** תלמיד (עם מקצועות ומבחנים), כללי (משימות בלבד), וקבוצה (משימות משותפות עם שיוך אנשים)
* **📚 ניהול מקצועות מותאם אישית:** הוספת מקצועות עם בחירת צבעים ייחודית לזיהוי ויזואלי מהיר
* **✅ מעקב משימות חכם:** הוספת שיעורי בית עם כותרת, תיאור, תאריך התחלה/הגשה, עדיפות, קבצים מצורפים ושדות מותאמים
* **📝 ניהול מבחנים מקיף:** מבחנים, בוחנים, בוחנים פתע; תאריכים מרובים (מועד א׳/ב׳/ג׳); ניהול ציונים, בונוס, תיקונים; נושאים ללימוד
* **📊 סטטיסטיקה בזמן אמת:** לוח בקרה עם סך המשימות, משימות שהושלמו, ממתינות, דחופות, מבחנים קרובים ומערכת XP
* **🏆 מערכת הישגים:** תגים וביצועים לדברים כמו "יום מושלם", "סטריק" וסיכום דיוקים
* **🔔 מערכת התראות ויזואלית:** סימון אוטומטי של משימות דחופות (פחות מ-48 שעות) או משימות באיחור; התראות דפדפן
* **📅 שילוב Google:** חיבור Google Tasks & Calendar לצפייה בתאריכים קרובים; שילוב Google Classroom לייבוא משימות
* **🎨 ערכות נושא ומצב לילה:** בחירה בין ערכות נושא מוגדרות מראש, מצב לילה אוטומטי, תמיכה בהעדפות המערכת
* **💾 סנכרון ענן:** כל הנתונים מסונכרנים ל-Firestore עם גיבוי אוטומטי; עבודה אופליין עם סנכרון בחזרה
* **📤 ייצוא נתונים:** ייצוא ל-JSON, PDF ו-Excel; ייבוא נתונים מקודמים

## 🛠 טכנולוגיות

הפרויקט נבנה כ-Single Page Application (SPA) מודרנית תוך שימוש בטכנולוגיות הבאות:

* **React 18 + Vite:** אפליקציה רכיבית בעלת ביצועים גבוהים עם dev server מהיר
* **Firebase:** אימות משתמשים (Email, Google, Phone) וסנכרון נתונים בזמן אמת דרך Firestore
* **HTML5 & CSS3:** עיצוב רספונסיבי ותומך RTL (עברית) בשימוש ב-Flexbox/Grid ומשתנים CSS
* **Lucide Icons:** ספריית אייקונים מודרנית למערכת
* **LocalStorage API:** אחסון מקומי לגיבוי מהיר והשבתה אופליין

## 🚀 התחלה מהירה

### דרישות
- Node.js 16+ ו-npm או yarn
- חשבון Firebase (לשימוש בענן)

### הגדרה מקומית
```bash
# התקן תלויות
npm install

# הפעל שרת פיתוח
npm run dev
# האתר נפתח ב-http://localhost:3000

# בנה לייצור
npm run build

# הצג גרסה מייצור מקומית
npm run preview

# העלה ל-GitHub Pages
npm run deploy
```

### הגדרה של Firebase
1. צור פרויקט חדש ב-[Firebase Console](https://console.firebase.google.com/)
2. אפשר Email/Password ו-Google Sign-In בחלק ה-Authentication
3. צור קובץ `firebase-config.js` עם הערכים שלך:
   ```javascript
   import { initializeApp } from 'firebase/app';
   import { getAuth } from 'firebase/auth';
   import { getFirestore } from 'firebase/firestore';

   const firebaseConfig = {
     apiKey: "YOUR_API_KEY",
     authDomain: "your-app.firebaseapp.com",
     projectId: "your-project-id",
     storageBucket: "your-bucket.appspot.com",
     messagingSenderId: "your-sender-id",
     appId: "your-app-id"
   };

   const app = initializeApp(firebaseConfig);
   export const auth = getAuth(app);
   export const db = getFirestore(app);
   ```
4. הגדר כללי Firestore ב-`firestore.rules`

## 📖 איך להשתמש?

### עבור תלמידים (מצב Student)
1. **בחר מצב:** הגדרות → בחר "מצב תלמיד" (כולל מקצועות ומבחנים)
2. **הוסף מקצועות:** לחץ "מקצועות" בתפריט, הוסף שם ובחר צבע
3. **צור משימה:** לחץ "+ הוסף משימה", מלא פרטים, בחר מקצוע ותאריך הגשה
4. **צור מבחן:** לחץ "+ הוסף מבחן", הוסף תאריך(ים), נושאים, משקל
5. **עקוב ציונים:** לאחר המבחן, הוסף ציון בחלון הזנת הציונים
6. **סמן הושלם:** לחץ על המשימה/מבחן וסמן "✓" בסיום
7. **בדוק סטטיסטיקות:** בדוק את לוח הבקרה למספרים בזמן אמת, או לחץ על "הישגים וסטטיסטיקות"

### עבור משימות כלליות (מצב General)
- משימות ללא מקצועות ומבחנים
- אידיאלי למטלות עבודה, פרויקטים אישיים וכדו׳

### עבור קבוצות (מצב Group)
- הוסף חברים בהגדרות
- צור משימות והקצה לאנשים ספציפיים
- עקוב אחר בעלות ומצב משימה

### תכונות נוספות
- **חיפוש:** Ctrl+F (או Command+F ב-Mac)
- **תמא לילה:** Ctrl+D או לחץ על אייקון הירח
- **סנכרון Google:** התחבר ל-Google Tasks & Calendar בהגדרות לצפייה בתאריכים
- **ייצוא:** הגדרות → "ייצא ל-JSON/PDF/Excel"
- **ייבוא:** הגדרות → "ייבא נתונים" משמורת קודמת
